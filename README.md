# 📚 DOC_EstagJCA - SpaceDesk E-bus

Repositório de documentação do **SpaceDesk E-bus**!

## Sobre este Repositório
Este repositório é dedicado exclusivamente para armazenar e versionar todas as documentações, levantamentos de requisitos e diagramas relacionados ao desenvolvimento do projeto de estágio no **Grupo JCA (2026)**.

## O Projeto
O **SpaceDesk E-bus** é um Sistema Web de Gestão de Espaços focado em organizar o acesso aos recursos físicos do escritório (2º e 13º andares). A aplicação permite a reserva de mesas, agendamento de salas de reunião com base na capacidade, e conta com check-in via QR Code para mitigar o efeito de "salas fantasmas" (No-Show).

## Estrutura de Arquivos
* 📄 **`RascunhoInicial`**: Visão geral do escopo, problema, objetivos e métricas de sucesso.
* 📄 **`LevantamentoRefinamentoRequisitos`**: Detalhamento técnico, User Stories (separadas por MVP e V2), Requisitos Não-Funcionais e Casos de Borda.

## Controle de Versão (Changelog)
A cada grande atualização na documentação, criação de diagramas (MER/DER) ou protótipos, a versão dos arquivos e deste repositório será incrementada (ex: v1.0 para v1.1).
